# Персональные приглашения — инструкция

## Как это работает

Каждому гостю выдаётся свой **ключ** — короткий случайный код.
Когда гость открывает ссылку с этим ключом, на сайте показывается личное
обращение к нему.

Ключи и обращения хранятся в файле `guests.js`.

## Ссылка для гостя

Базовый адрес сайта:

```
https://mtsarevich.github.io/
```

Чтобы получилось персональное приглашение, добавьте в конец `?to=КЛЮЧ`:

```
https://mtsarevich.github.io/?to=22ez8x
```

Если сайт опубликован на другом адресе — замените начало ссылки на свой адрес,
часть `?to=КЛЮЧ` остаётся такой же.

## Шаг 1. Впишите обращения

Откройте `guests.js` и впишите обращение в кавычки напротив каждого ключа:

```js
var GUESTS = {
  "22ez8x": "Иван",
  "2m2mjp": "Семья Петровых",
  "32bpzq": "Александра Николаевна",
  ...
};
```

- Обращение — это текст, который увидит гость (имя, «Дорогие друзья»,
  «Семья Ивановых» и т. д.).
- Сами ключи менять не нужно — они уже случайные.
- Ключ с пустым обращением (`""`) просто покажет сайт без личного обращения.

## Шаг 2. Раздайте ссылки

Для каждого гостя возьмите его ключ и составьте ссылку `?to=КЛЮЧ`.
Учёт удобно вести в файле `гости.csv` (открывается в Numbers, Excel,
Google Таблицах):

- **Обращение** — как обращаемся к гостю;
- **Кому отправляю** — заметка для себя (кому именно эта ссылка);
- **Ключ** — код для ссылки.

Обращение в `гости.csv` и в `guests.js` для одного ключа должно совпадать:
таблица — для вашего удобства, а сайт читает именно `guests.js`.

## Как добавить новый ключ (сверх имеющихся 34)

1. Придумайте новый код: 6 символов, маленькие латинские буквы и цифры,
   например `k7m2x9`. Не используйте уже занятые коды.
2. Добавьте строку в `guests.js` (не забудьте запятую в конце предыдущей строки):
   ```js
   "k7m2x9": "Имя гостя"
   ```
3. Ссылка для него: `https://mtsarevich.github.io/?to=k7m2x9`

## Важно

- Ключи чувствительны к регистру — пишите их маленькими буквами.
- После правок `guests.js` изменения нужно опубликовать (закоммитить и
  запушить в репозиторий), иначе на сайте они не появятся.
